# 快速開始

歡迎使用 Onlysaid！本指南將幫助您快速上手，開始使用自然語言與智能體協作完成任務。

## 🚀 第一步：建立帳戶

1. **造訪官網**：前往 [onlysaid.com](https://onlysaid.com)
2. **註冊帳戶**：點擊「註冊」按鈕
3. **驗證電子郵件**：檢查您的收件匣並點擊驗證連結
4. **完成設定**：填寫基本資料並選擇您的偏好設定

## 🏢 建立您的第一個工作區

工作區是您和團隊協作的中心。以下是建立步驟：

### 建立工作區

1. 登入後，點擊「新建工作區」
2. 輸入工作區名稱（例如：「我的專案」）
3. 新增描述（可選）
4. 選擇工作區類型：
   - **個人**：適合個人專案
   - **團隊**：適合小型團隊協作
   - **企業**：適合大型組織

### 邀請團隊成員

1. 在工作區設定中點擊「邀請成員」
2. 輸入團隊成員的電子郵件地址
3. 設定權限等級：
   - **檢視者**：只能檢視內容
   - **編輯者**：可以建立和編輯內容
   - **管理員**：可以管理成員和設定
4. 發送邀請

## 🤖 認識智能體

智能體是您的 AI 助手，每個都有特殊技能：

### 主要智能體類型

- **📊 分析師**：數據分析和報告生成
- **💻 開發者**：程式碼審查和技術支援
- **📝 寫手**：內容創作和編輯
- **🎨 設計師**：視覺設計和創意工作
- **📋 專案經理**：任務規劃和進度追蹤

### 如何與智能體互動

只需使用自然語言描述您想要完成的任務：

```
「請分析上個月的銷售數據並生成報告」
「幫我審查這段 Python 程式碼的效能」
「為新產品發佈會寫一份新聞稿」
「設計一個現代風格的標誌」
```

## 📁 檔案管理

### 上傳檔案

1. **拖拽上傳**：直接將檔案拖拽到工作區
2. **點擊上傳**：使用上傳按鈕選擇檔案
3. **雲端同步**：連接 Google Drive、Dropbox 等

### 支援的檔案類型

- **文件**：PDF、Word、PowerPoint、Excel
- **程式碼**：所有主要程式語言
- **圖片**：PNG、JPG、SVG、GIF
- **數據**：CSV、JSON、XML

### 檔案組織

- 使用資料夾結構組織檔案
- 新增標籤便於搜尋
- 設定檔案權限控制存取

## 💬 開始您的第一次對話

### 基本對話範例

**任務規劃**：

```
「我需要為下個月的產品發佈制定一個專案計劃」
```

**數據分析**：

```
「請分析這個 CSV 檔案中的客戶數據，找出主要趨勢」
```

**程式碼協助**：

```
「幫我優化這個 JavaScript 函數的效能」
```

**內容創作**：

```
「為我們的新功能寫一篇部落格文章」
```

### 進階技巧

- **具體描述**：提供詳細的背景資訊
- **分步驟**：將複雜任務分解為小步驟
- **提供範例**：給出期望結果的範例
- **設定截止日期**：告知時間要求
